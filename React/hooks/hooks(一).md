# React hooks 的使用

### Hooks

hooks 可以在我们不编写class的情况下使用state以及其他的React特性。并且hook只能在函数式组件中使用。hooks的让函数式组件也可以进行状态管理。

**基础Hook**

- useState
- useEffect
- useContext

**额外的Hook**

- useReducer
- useCallback
- useMemo
- useRef
- useImperativeHandle
- useLayoutEffect
- useDebugValue

在react开发中class组件的痛点：

1. class组件代码复杂，随着业务代码的不断增加。class组件会变得越来越复杂难以管理
2. 组件状态难以复用，在class组件中使用context会变得复杂。容易出现嵌套过多的情况。
